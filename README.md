# NFS-Heat-Modding-PS4

A community project dedicated to exploring and documenting Need for Speed Heat modding on PlayStation 4. This repository covers the techniques, tools, and research behind modifying game and save data to unlock features that aren't normally possible.

What is Possible? (For Now)
Car Part Removal – Remove or hide body parts such as bumpers, hoods, spoilers, mirrors, fenders, and more.
Engine Swaps – Install different engines into vehicles to create unique performance builds.
Car Swaps – Replace one vehicle with another, allowing you to drive cars outside their original garage slot.
Money Editing – Modify your in-game cash balance for testing or custom save setups.
Save Editing – Edit various garage and progression values where supported.
Purpose

This repository is intended to document discoveries, tools, tutorials, and research related to PS4 NFS Heat modding. Whether you're interested in customizing vehicles beyond the game's limits or learning how the game's data works, you'll find guides and resources here as the project grows.

Disclaimer: This project is for educational and research purposes only. Always create a backup of your save data before making any modifications. Use modified saves at your own risk.
